# DSP Notebooks
Teaching notebooks for a transitional Digital Signal Processing course.

This repository contains teaching notebooks for an introductory/transitional Digital Signal Processing course, with emphasis on practical understanding, visualization, and classroom/lab activities.

These notebooks are not intended to be a complete DSP textbook. Mathematical derivations are intentionally brief, and some topics are introduced through computational experiments before formal theoretical treatment.

## Intended Audience

The materials are designed for students who have basic programming experience and some prior exposure to signals and Fourier analysis.

## Status

These materials are under active classroom use and revision. Some explanations may be simplified for teaching purposes.
